# Competitive Analysis: Fertility Tracking Apps

## Market Overview

The fertility tracking market is valued at $1.1B (2024) and growing at 8% CAGR. Key players focus on either mass-market appeal OR power-user features, leaving a gap for unified intelligence.

---

## Direct Competitors

### 1. Fertility Friend (FF)
**Website:** fertilityfriend.com  
**Pricing:** Free / $55/year premium  
**Downloads:** 5M+

| Strengths | Weaknesses |
|-----------|------------|
| Most data-rich charting | Dated, confusing UI |
| Strong community | Steep learning curve |
| Multiple chart types | No multi-app reconciliation |
| Research-backed | Overwhelming for beginners |

**Why users leave:** "The app looks like it was designed in 2005"

---

### 2. Clue
**Website:** helloclue.com  
**Pricing:** Free / $40/year premium  
**Downloads:** 12M+

| Strengths | Weaknesses |
|-----------|------------|
| Clean, modern UI | Predictions often inaccurate |
| Gender-inclusive | Limited charting depth |
| Privacy-focused | No BBT integration |
| Easy to use | Algorithm is a black box |

**Why users leave:** "It told me I was fertile when I clearly wasn't"

---

### 3. Flo
**Website:** flo.health  
**Pricing:** Free / $50/year premium  
**Downloads:** 200M+

| Strengths | Weaknesses |
|-----------|------------|
| Largest user base | Sells data to Facebook |
| AI predictions | Privacy scandals |
| Content library | Mass-market, not TTC-focused |
| Partner app | Inaccurate for irregular cycles |

**Why users leave:** "I don't trust them with my data after the FTC settlement"

---

### 4. Natural Cycles
**Website:** naturalcycles.com  
**Pricing:** $100/year  
**Funding:** $100M+ raised

| Strengths | Weaknesses |
|-----------|------------|
| FDA cleared | Expensive |
| Medical device status | Algorithm-only (no biomarker verify) |
| Wearable integration | Paywalls Samsung watch data |
| Strong brand | Not method-compliant (FAM) |

**Why users leave:** "It gave me a green day when my CM said otherwise"

---

### 5. Premom
**Website:** premom.com  
**Pricing:** Free / $20/year  
**Downloads:** 3M+

| Strengths | Weaknesses |
|-----------|------------|
| OPK photo analysis | Inconsistent readings |
| Affordable | Chinese data privacy concerns |
| Easy to use | Limited charting |
| Community features | No multi-signal verification |

**Why users leave:** "The OPK reader gives different results for the same strip"

---

### 6. Read Your Body (RYB)
**Website:** readyourbody.com  
**Pricing:** £19/year ($24)  
**Target:** FAM practitioners

| Strengths | Weaknesses |
|-----------|------------|
| Method-agnostic | No prediction (chart-only) |
| Educator directory | No partner sync |
| Export-friendly | Small user base |
| Privacy-first | No algorithm assistance |

**Why users leave:** "I want something that tells me what the chart means"

|-----------------|---------------|

## Key Insight

**No competitor reconciles data from multiple apps.**

Users are already tracking in 2-4 apps. They want ONE answer, not more predictions. Fertile is the only tool that takes their existing data and synthesizes it.
